COMMENT OUVRIR LE SITE EN LOCAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Placez index.html, style.css et le dossier assets/ dans le même répertoire.
2. Ouvrez index.html dans votre navigateur (double-clic).
3. Les images s'afficheront dès qu'elles sont placées aux bons chemins.
4. Sans images, le layout reste intact — les balises img occupent leur espace en attendant.
5. Pour le développement : utilisez VS Code + extension "Live Server" pour le rechargement automatique.
